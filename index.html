A free (as in freedom) clone of the Age of Empires II engine

View the Code Download .tar.gz

You know Age of Empires. You know its bugs. You know the fun playing it.

We are creating a free (as in freedom) clone of the Age of Empires II engine in C++17 from scratch (GPLv3+). We use "modern" technologies like OpenGL/GLSL, Python, Qt5, CMake, ...

We aim to create an exact clone (for now), using the original media files.

This allows people using GNU/Linux, BSD, macOS or Windows to play the game natively, minus features, plus additional epic bugs (or was it vice versa?).

The project is run by volunteers: Students and professionals from all over the world.

Join us and help to create an awesome game!
