Free cross-platform RTS game engine based on Age of Empires mechanics

Repository Windows Packages Linux Packages

You know Age of Empires. You know its bugs. You know the fun playing it.

We are creating a free (as in freedom, GPLv3+) cross-platform RTS game engine based on Age of Empires mechanics in C++17. We use "modern" technologies like OpenGL/GLSL, Python, Qt5, CMake, ...

This allows people using GNU/Linux, BSD, macOS or Windows to play the game natively.

We aim to create a game engine that provides the original look and feel. Until we have free replacement assets, we can convert original media files provided by the user to an format compatible to openage.

A look into the future: openage has a lot of improvements planned. It is going to add completely new possibilities and will provide better performance due to different design decisions being made from the beginning on, e.g. being multithreaded and using modern technologies like Vulkan rendering. Our self-designed entity-component-system nyan will help us exposing a Modding API making openage more easy for modders to use features of openage in their mods.

We always welcome new contributors! openage is a huge and ambitious project. There is more work than our small core dev team mainly run by volunteers can handle. The openage community is very open, welcoming and supportive. If you’ve got the enthusiasm, you’re all welcome.

Get in contact with us if you have any issues getting started.
